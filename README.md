# CS780 Capstone Project

This repository contains my implementation and experiments for the CS780 capstone project.

## Overview

This project explores training and improving agents using multiple approaches. Different training strategies were implemented and compared, with results saved for analysis and submission.

## Repository Structure

* **agent.py**
  Contains the final trained agent used for evaluation and submission.

* **train_*.py files**
  These files correspond to different training methods and approaches that were tried during the project.

* ***.pth files**
  Each training script has its corresponding saved model weights.

* **submissions/**
  This folder contains all the zip files submitted on Codabench.
  FINAL SUBMISSION IS NAMED W3_7.ZIP
  W2_6 IS THE SUBMISSION I SUBMITTED AS IT GAVE HIGHEST SUCCESS RATE
  
  They are organized week-wise and include the agent along with its corresponding weights.

## Notes

* The repository includes multiple experimental approaches, but `agent.py` represents the final selected solution.
* Training scripts are kept for reference and reproducibility of results.
* Saved weights allow loading and testing different trained versions without retraining.
